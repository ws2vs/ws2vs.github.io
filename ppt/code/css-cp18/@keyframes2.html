不支持动画的属性将被忽略

                
@keyframes fadeout-invalid {
    from {
        display: none;
    }
    to {
        display: block;
    }
}

不支持动画但不会被忽略的属性

                
@keyframes animate-no-middle-special {
    from {
        visibility: visible;
    }
    to {
        visibility: hidden;
    }
}

属性值没有中间值

                
@keyframes animate-no-middle {
    0% {
        width: auto;
    }
    100% {
        width: 300px;
    }
}
